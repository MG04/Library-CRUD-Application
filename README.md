# Library Management 

Simple CRUD application made for a university course.

## Technologies used
- [React](https://react.dev/)
- [Nodejs](https://nodejs.org)
- [MySQL](https://www.mysql.com)

## Getting Started

### Prerequisites
- [Nodejs](https://nodejs.org/en/download)
- [Prisma](https://www.prisma.io/)

The above should be installed.

Follow the commands below to kickstart the application.
```shell
git clone https://github.com/MG04/UniWa-Library
cd UniWa-Library
```

Now install the dependencies
```shell
cd backend
npm install
npm install @prisma/client --save
npx prisma generate
```
Open another terminal in folder.
```shell
cd frontend
npm install
npm install @prisma/client --save
```

We are almost done, Now just start the development server.

For Backend.
```shell
cd backend
npm start
```
For Frontend.

Open another terminal in folder.
```shell
cd frontend  
npm start
```
Done! Now open localhost:3000 in your browser.
